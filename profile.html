Stages

Q-Aviation Yisual

Bij Q-Aviation heb ik mijn eerste stage gedaan aan het eind van 2018, deze duurde in totaal 10 weken.
Q-Aviation specialiseert in het maken van de lichten voor helidecks en helipads. Ik heb voor hun een calculator gemaakt voor het uitreken van hoeveel lichten een helideck of pad nodig heeft en dit visueel weergeven.
Hierdoor krijgen de klanten een visueel overzicht van hoe hun helideck of pad eruit gaat zien door middel van een paar dimensies in te invoeren, ook weten ze gelijk hoeveel ze moeten bestellen bij Q-Aviation.

Q-Aviation Meer over "helideck calculator"

Yisual was mijn tweede stage, hier liep ik stage van augustus 2019 tot januari 2020 en duure 20 weken.
Yisual specialiseert in het desingen van brochures voor de huizen die makenlaars verkopen.
Mijn opdracht was om een API te maken die bestellingen die klanten bij Yisual plaatsen door te sturen naar de gekozen drukker.

Yisual Meer over "Press-Print"

Projecten

Helideck Calculator Press-Print VOS Pasimo

Dit was mijn project tijdens mijn stage bij Q-Aviation, het doel van het project was om klanten makkenlijker te kunnen laten zien hoeveel lampen ze voor hun helideck/pad nodig hebben.
De klant vult de dimensies van hun helidack/pad in en kiest uit een paar opties, aan de hand daarvan wordt berekend hoeveel van welke producten ze nodig hebben. Daarnaast ziet de klant het ook visueel door middel van een image overlay. Tijdens dit project heb ik voornamelijk gebruik gemaakt van PHP, JS, HTML en CSS, De reden waarom ik deze talen heb gebruikt is omdat de rest van hun website ze ook gebruikt.

Helideck/pad calculator

Press-Print is de naam van het prject dat ik tijdens mijn stage bij Yisual heb gedaan, Press-Print is een API dat bestellingen van brochures automatish doorstuurd naar de gekozen drukker van die klant die Yisual ondersteund. Het lastige hieraan was dat ik een universeel Yisual product moest maken wat dan omgezet wordt in de juiste format die de gekozen drukker gebruikt.
voor dit project heb ik voornamelijk JavaScript en TypeScript gebruikt, ook heb ik veel geleerd over het documenteren van code en het automatish laten opstellen van documentatie.

Tijdens de eerste 2 jaar van mijn MBO opleiding werkte ik aan het project VOS, het staat voor "Vinger Opsteek Systeem".
Tijdens het eerste jaar heb ik verder gewerkt aan het bestaande vos wat gebruikt maakte van Java, Spring Boot, HTML, CSS en nog een paar talen. toen ik na mij stage in de tweede weer aan VOS moest gaan werken kregen we te horen dat we opnieuw moesten begginnen omdat het oude VOS te groot was geworden. De bedoeling was dat de student online via zijn windows task bar makkenlijk zijn vinger kon opsteken en dat de docent dan in zijn scherm een melding kreeg dat iemand een vraag heeft. Voor de VOS reboot hebben we dezelfde talen gebruikt alleen implaats van Spring Boot moesten we Struts2 gebruiken.

Pasimo was mijn eidproject voor het MBO met alle adnere examenstudenten. Pasimo is een applicatie om de aanwezigheid van studenten en docenten bij te kunnen houden. Aan het begin van de dag inchecken met je Scalda pas, tijdens pauzes uitcheken enz.
Het is het grootste project van de opleiding en ik heb gewerkt aan het kunnen aanmaken van schooljaren.
Het was meer werk dan ik dacht want schooljaren hebben met een heleboel te maken. Voor Pasimo hebben we gewerkt met Java, Struts2, HTML, CSS en nog een paar talen.

HZ Mijn opleiding

Aan het Scalda heb ik de opleiding "software-developer" gevolgd en ben hier in 2020 voor geslaagd.
Het is een 3 jarige opleiding waar je in de eerste helft van het eerste jaar les krijgt over prgrameren de rest van het jaar zit je in projecten met gemixde jaren.
In het tweede jaar hem ik een stage van 10 weken gedaan bij Q-Aviation waar ik een helideck calculator heb gemaakt, de rest van het jaar zit je in projecten.
tot slot het laatste jaar heb ik een stage van 20 weken gedaan bij Yisual en heb ik een API gemaakt, als je terug om school bent wordt je met de rest van de examenkandidaten in een examenproject geplaats om examen te doen
