Hierdoor krijgen de klanten een visueel overzicht van hoe hun helideck of pad eruit gaat zien door middel van een paar dimensies in te invoeren, ook weten ze gelijk hoeveel ze moeten bestellen bij Q-Aviation.

Projecten

Helideck Calculator Press-Print VOS Pasimo

Dit was mijn project tijdens mijn stage bij Q-Aviation, het doel van het project was om klanten makkenlijker te kunnen laten zien hoeveel lampen ze voor hun helideck/pad nodig hebben.
De klant vult de dimensies van hun helidack/pad in en kiest uit een paar opties, aan de hand daarvan wordt berekend hoeveel van welke producten ze nodig hebben. Daarnaast ziet de klant het ook visueel door middel van een image overlay. Tijdens dit project heb ik voornamelijk gebruik gemaakt van PHP, JS, HTML en CSS, De reden waarom ik deze talen heb gebruikt is omdat de rest van hun website ze ook gebruikt.

Helideck/pad calculator

Press-Print is de naam van het prject dat ik tijdens mijn stage bij Yisual heb gedaan, Press-Print is een API dat bestellingen van brochures automatish doorstuurd naar de gekozen drukker van die klant die Yisual ondersteund. Het lastige hieraan was dat ik een universeel Yisual product moest maken wat dan omgezet wordt in de juiste format die de gekozen drukker gebruikt.
voor dit project heb ik voornamelijk JavaScript en TypeScript gebruikt, ook heb ik veel geleerd over het documenteren van code en het automatish laten opstellen van documentatie.

Tijdens de eerste 2 jaar van mijn MBO opleiding werkte ik aan het project VOS, het staat voor "Vinger Opsteek Systeem".
Tijdens het eerste jaar heb ik verder gewerkt aan het bestaande vos wat gebruikt maakte van Java, Spring Boot, HTML, CSS en nog een paar talen. toen ik na mij stage in de tweede weer aan VOS moest gaan werken kregen we te horen dat we opnieuw moesten begginnen omdat het oude VOS te groot was geworden.
